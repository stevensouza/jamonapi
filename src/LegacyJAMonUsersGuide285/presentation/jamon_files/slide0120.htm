Monitoring Web Applications
lIn Web applications it is particularly important to monitor code, due to
lAn unpredictable load and number of users
lMany possible bottlenecks:  Is the performance problem in servlets, JSPs, EJBs, the web server, the database, the network…?
lBy building monitoring into an application from the start, developers can spend time solving performance problems not diagnosing them!
